## Evidence Grades
- **A:** Org's own website/letter says "donate through CFI"
- **B:** IRS filing, FOIA response, court filing names CFI as funder
- **C:** Investigative journalism documents the link
- **D:** Media report mentions in passing

## Status Key
- `pending` — not yet researched
- `searching` — research in progress
- `found` — CFI connection found, evidence captured
- `not-found` — searched thoroughly, no primary source found
- `removed` — evidence exists in Wayback only (org removed CFI references)
- `not-cfi` — org uses a different US fiscal sponsor, not CFI
